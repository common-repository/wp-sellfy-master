A simple plugin that makes it easy to webpage connect with the Sellfy.com

== Description ==

Sellfy.com is the new project which allow selling our digital goods. It's a great solution for WordPress users which would like to start selling their files.

This plugin give you some tools to fast presenting your Sellfy offers on your webpage based on WordPress. 

Features:

*	A widget presenting one offer
*	Automatic loading of offer's title in widget
*	A shortcode [sellfy type="SMALL/LARGE"]OFFER_URL[/sellfy]
*	You can set button size in each widget and shortcode

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use widget or shortcode
